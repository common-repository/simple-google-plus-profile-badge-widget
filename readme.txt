This plugin allows you to set new Google Plus badge widget to your WordPress site.

== Description ==

Simple google plus badge widget allows you to quickly and easily add the new Simple Google Plus Profile Badge Widget to your website without need to touch css or any code. Just add page name or profile id.

Plugin by <a href="http://saucerweb.com">Web Design Company Bangladesh</a>

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `simple-google-plus-profile-badge-widget` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the Simple Google Plus Profile Badge Box widget from the widgets window in WordPress' admin panel.
4. Configure the available options with your page type, URL, desired style and size.
